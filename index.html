Questões de PWI

1 - Java Script e Java são a mesma coisa?

Não, JavaScript e Java são linguagens de programação diferentes, com propósitos distintos.

JavaScript

  • Finalidade: Principalmente usada para desenvolvimento web, adicionando interatividade a páginas web.
  • Ambiente de Execução: Executa principalmente no navegador (client-side), mas também pode ser usado no servidor com Node.js.
  • Sintaxe: Flexível, com tipagem dinâmica.
  • Uso Comum: Manipulação de DOM, criação de eventos, validação de formulários, animações.
  • Popularidade: Essencial para desenvolvimento web frontend.
Java
  • Finalidade: Usada para desenvolvimento de aplicativos, especialmente aplicativos empresariais, Android, e sistemas de backend.
  • Ambiente de Execução: Executa na JVM (Java Virtual Machine), tornando-o portátil entre diferentes plataformas.
  • Sintaxe: Estruturada, com tipagem estática.
  • Uso Comum: Aplicativos de desktop, aplicativos Android, sistemas de backend robustos.
  • Popularidade: Amplamente usada em ambientes empresariais e desenvolvimento de aplicativos Android.

2 - Quando surigu o Java Script?

O JavaScript foi criado em 1995. A linguagem foi desenvolvida em apenas 10 dias com o objetivo de simplificar a validação de formulários em páginas web.

3 - Quem foi o responsável pela criação do Java Script?

Ele foi criado por Brendan Eich enquanto ele trabalhava na Netscape Communications Corporation.

4 - Java Script e Java são a mesma coisa? Qual o caminho trilhado pelo reponsável pela criação do Java Script, trata-se de alguma empresa ainda no mercado?

Não, JavaScript e Java são linguagens de programação diferentes, com propósitos distintos.
Brendan Eich, o criador do JavaScript, teve uma carreira muito sólida. Após criar o JavaScript enquanto trabalhava na Netscape Communications Corporation, ele co-fundou o projeto Mozilla em 1998. Mais tarde, ele se tornou o CTO e CEO da Mozilla Corporation. Em 2014, ele se demitiu da Mozilla devido a controvérsias relacionadas às suas opiniões pessoais. Em 2015, Eich co-fundou a Brave Software, uma empresa que desenvolve o navegador Brave, focado em privacidade e velocidade. A Brave Software ainda está no mercado e continua a evoluir. A Netscape Communications Corporation, por outro lado, foi adquirida pela AOL em 1999 e encerrou suas operações em 2008. No entanto, a marca Netscape ainda é usada pela AOL para oferecer serviços de internet descontinuados.
